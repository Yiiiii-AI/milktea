## 项目概述
奶茶博物馆网站，一个展示奶茶文化的静态网页站点。

## 技术栈
- 纯静态 HTML/CSS/图片
- 无构建工具
- 无 JavaScript 框架

## 目录结构
```
project/
├── index.html          # 首页
├── map.html            # 奶茶地图页
├── news.html           # 奶茶资讯页
├── pairing.html        # 奶茶搭配页
├── ranking.html        # 奶茶榜单页
└── static/
    ├── css/            # 样式文件
    └── images/         # 图片资源
```

## 关键入口 / 核心模块
- 首页入口：`project/index.html`
- 页面导航：首页、奶茶搭配、奶茶榜单、奶茶资讯、奶茶地图

## 运行与预览
- 预览命令：`npx serve project -l 5000`
- 直接用浏览器打开 HTML 文件也可查看

## 用户偏好与长期约束
- 纯静态项目，无需构建流程
- 所有页面引用使用相对路径

## 常见问题和预防
- 预览时确保使用静态文件服务器，避免跨域问题
